# WordPress セットアップ完了チェックリスト

セットアップ完了後にこのチェックリストで確認を行う。

## 基本設定

- [ ] サイトタイトルが正しく設定されている
- [ ] キャッチフレーズが設定されている
- [ ] タイムゾーンが Asia/Tokyo に設定されている
- [ ] パーマリンク構造が「投稿名」(`/%postname%/`) に設定されている
- [ ] 日付・時刻形式が適切に設定されている

## コンテンツ

- [ ] デフォルトの「Hello world!」投稿が削除されている
- [ ] デフォルトの「Sample Page」が削除されている
- [ ] 必要なページが作成されている
- [ ] フロントページが固定ページに設定されている
- [ ] 投稿ページが指定されている

## テーマ

- [ ] テーマのスタイルバリエーションが適用されている
- [ ] ブランドカラーが反映されている

## カテゴリ・メニュー

- [ ] 必要なカテゴリが作成されている
- [ ] デフォルトの「未分類」カテゴリが整理されている
- [ ] ナビゲーションメニューが正しく構成されている
- [ ] メニューリンクが正しいページを指している

## プラグイン

- [ ] 必要なプラグインがインストール・有効化されている
- [ ] WP Multibyte Patch が有効（日本語サイトの場合）
- [ ] SEOプラグインが有効（必要な場合）

## REST API

- [ ] アプリケーションパスワードが発行されている
- [ ] REST API で投稿の取得が成功する
- [ ] `.env` ファイルに認証情報が保存されている（必要な場合）

## セキュリティ

- [ ] 管理者パスワードが強固である
- [ ] アプリケーションパスワードが安全に保管されている
- [ ] 不要なデフォルトコンテンツが削除されている
